# Système de Gestion de Clients

## Description
Application professionnelle de gestion de clients développée en Python
avec une interface graphique Tkinter et une base de données MySQL.

Ce projet permet de gérer jusqu’à 10 000 clients de manière sécurisée.

## Fonctionnalités
- Authentification sécurisée (Admin / Utilisateur)
- Ajouter un client
- Modifier un client
- Supprimer un client
- Rechercher un client
- Exporter les données des clients en PDF
- Interface graphique moderne avec logo
- Gestion de rôles utilisateurs

## Technologies utilisées
- Python
- Tkinter
- MySQL
- Git & GitHub

## Sécurité
- Accès par identifiant et mot de passe
- Gestion des rôles (Administrateur / Utilisateur)
